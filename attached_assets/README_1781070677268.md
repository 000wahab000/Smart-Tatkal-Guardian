# Smart Tatkal Guardian — Simulator Service

> Person 1's service. Generates fake IRCTC booking traffic: 80% bots, 20% humans.

Part of the **Smart Tatkal Guardian** multi-service bot detection system built for FAR AWAY 2026 Hackathon by Team AlgoMinds.

## Setup

```bash
cd simulator
pip install -r requirements.txt
```

## Run

```bash
uvicorn main:app --port 8001 --reload
```

## Endpoints

| Method | Path | Description |
|---|---|---|
| POST | `/simulate/start?rps=10` | Start generating traffic (default 10 req/sec) |
| POST | `/simulate/stop` | Stop generation |
| GET | `/simulate/status` | Live counters: running, sent, bots, humans |
| GET | `/simulate/replay` | Last 100 requests (internal — includes `is_bot`) |

## Architecture Notes

- Sends to `http://localhost:8000/ingest` (orchestrator service)
- `is_bot` field is **stripped** in `sender.py` before sending — internal tracking only
- 80% bot traffic, 20% human traffic
- Bot profiles: shared IPs, repeated fingerprints, 50-200ms intervals, no mouse movement
- Human profiles: unique IPs, unique fingerprints, 3-15s intervals, realistic typing speed

```bash

cd scorer && uvicorn main:app --port 8002

# Terminal 2 — Orchestrator  
cd orchestrator && uvicorn main:app --port 8000

cd simulator && uvicorn main:app --port 8001 --reload

# Terminal 4 — Dashboard
cd dashboard && npm run dev
```
